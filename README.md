# 📉 ChurnGuard Pro

ChurnGuard Pro is an end-to-end churn prediction and customer intelligence application built using Streamlit and machine learning.

It is designed to move beyond prediction and help businesses understand **why customers churn and how to retain them**.

---

## 🚀 Features

### 🔍 Churn Prediction

* Machine learning model (Random Forest)
* Customer-level churn probability scoring
* Risk classification (High / Medium / Low)

### 📊 Interactive Dashboard

* Filter customers by risk level, contract type, and charges
* Visualize churn distribution and trends

### 🧠 Customer Insights

* Dynamic churn explanations for each customer
* Personalized retention recommendations

### 🛠️ Custom Chart Builder

* Create bar, line, scatter, histogram, and box plots
* Explore relationships across any features

### 🔄 What-if Simulator

* Modify customer attributes
* Instantly see how churn risk changes

### 💬 Churn Chatbot

* Ask natural questions like:

  * "Who are the highest risk customers?"
  * "What are the main churn drivers?"
  * "Why will this customer churn?"

* Python

* Pandas

---

## 📂 Dataset

This project uses a Telco Customer Churn dataset containing features such as:

* Contract type
* Tenure
* Monthly and total charges
* Payment method
* Service usage (internet, support, security)
* Churn label and churn reason

## ⚙️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/churnguard-pro.git
cd churnguard-pro
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the app

```bash
streamlit run churnapp.py
```
